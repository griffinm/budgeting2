# Budgeting App

Rails API backend for a personal finance/budgeting application. Connects to Plaid for bank account syncing and transaction imports. Supports multiple users per account, transaction categorization, merchant grouping, and financial dashboards.

## Tech Stack

- Ruby on Rails (API mode)
- PostgreSQL
- Plaid API for bank integration
- JWT authentication via `x-budgeting-token` header
- JBuilder for JSON response templates
